# DOOM on GitHub Pages

A minimal GitHub Pages site that runs classic DOOM in-browser using a WebAssembly-powered MS-DOS emulator embed.

## Live page

Once Pages is enabled:

`https://usegitange.github.io/doom-gh-pages/`

## Notes

- This project embeds the playable DOOM page hosted on Archive.org.
- If the iframe is blocked on some networks, use the direct Archive.org link shown on the page.
